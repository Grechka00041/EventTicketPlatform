# Event Ticket Platform

Платформа для управления событиями и продажи билетов, реализованная на **Spring Boot**. Система поддерживает полный цикл жизни события: от создания организатором до продажи билетов посетителям и валидации на входе через QR-коды.

Проект разработан с упором на чистую архитектуру, безопасность (OAuth2/OIDC) и отказоустойчивость (блокировки при конкурентной покупке).

## Функциональные возможности

### Для Организаторов (Organizers)
- Создание и редактирование событий (название, даты, место, статус).
- Управление типами билетов (цена, количество, описание).
- Просмотр списка своих событий с пагинацией.
- Удаление событий.

### Для Посетителей (Attendees)
- Просмотр опубликованных событий (публичный доступ).
- Поиск событий по названию и месту проведения (PostgreSQL Full-Text Search).
- Покупка билетов с генерацией уникального QR-кода.
- Личный кабинет с историей покупок.
- Просмотр деталей билета и скачивание QR-кода.

### Для Персонала (Staff)
- Валидация билетов на входе.
- Сканирование QR-кодов или ручной ввод ID билета.
- Защита от повторного использования билета (idempotency check).

## 🛠 Технологический стек

**Backend:**
- **Java 21** (LTS)
- **Spring Boot 3.4.4**
- **Spring Security** + **OAuth2 Resource Server**
- **Spring Data JPA** (Hibernate)
- **PostgreSQL** (Production DB) / **H2** (Testing)
- **MapStruct** (Mapping DTO <-> Entity)
- **Lombok** (Boilerplate reduction)
- **ZXing** (Генерация QR-кодов)

**Infrastructure & Tools:**
- **Docker & Docker Compose** (PostgreSQL, Keycloak, Adminer)
- **Keycloak** (Identity and Access Management)
- **Maven** (Build tool)
